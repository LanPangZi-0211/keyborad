#### 说明

1. Android 聊天/评论表情键盘 直接添加源码到项目使用方便自定义修改
2. 具体使用方法参见项目demo MainActivity
3. 需要增加自定义表情 仿照经典表情添加即可
4. 项目打包开启 shrinkResources true 则需添加res>>raw>>keep.xml 防止表情图片被移除
